1) install
npm install

2) start local node in a terminal
npm run node
(this runs a Hardhat local chain at http://127.0.0.1:8545)

3) compile (optional)
npm run compile

4) deploy (in a new terminal)
npm run deploy
(deployedAddress.txt will be created)

5) interact / watch / retrieve
npm run interact
npm run watch
npm run retrieve
